Show Me: Related Research Work

Goal of this project is to help the research community to find and understand related research work such as scholarly papers or articles, as well as to help each other in this process.

I am developing a Single Page RESTful Application using Vue.js which will facilitate features such as finding a new paper and its related research paper, visualize relationships in the form of graph, upvote any relationship, comment on any relationship, navigate through history of browsed papers. Project uses Node.js as backend application server, Vue.js as client application, MongoDB to store user information, Neo4j to store relationships as a graph database.
